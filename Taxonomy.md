Taxonomy of graph algorithms and graph data science techniques
```
Graph algorithms
└── Connectivity
    ├── Shortest path
    ├── Connected components
    └── Diameter
└── Centrality
    ├── Degree centrality
    ├── Betweenness centrality
    └── Eigenvector centrality
└── Community detection
    ├── Girvan-Newman algorithm
    ├── Louvain algorithm
    └── Label propagation algorithm
└── Other
    ├── Pathfinding
    ├── Graph isomorphism
    └── Graph matching

Graph data science techniques
└── Graph kernels
    ├── WL kernel
    ├── Shortest path kernel
    └── Edge kernel
└── Graph embedding
    ├── Deepwalk
    ├── Node2vec
    └── Graph convolutional networks (GCNs)
└── Graph visualization
    ├── Force-directed layout
    └── Graph drawing
```
